# Elite Points Academy - LP モックアップ

米国クレカ活用支援サービス「Elite Points Academy」のランディングページ（モック版）です。

## ページ構成

| パス | 内容 | 備考 |
|------|------|------|
| `/` | トップページ | LP1・LP2への導線（モック確認用表示付き） |
| `/lp1/` | LP1: 営業カタログ | パスワード認証あり（PW: `password`） |
| `/lp2/` | LP2: SNS広告用 | CV特化ページ |

## LP1 - 営業カタログ

- **ターゲット**: アラフォー経営者（年収2000万〜）
- **訴求**: 理性・納得（実績・再現性・仕組み）
- **デザイン**: 緑・ゴールド・ホワイト基調、余白広め、信頼感
- **構成**: Hero → Story → My Journey → Achievement → Voice(カルーセル) → Support → CTA
- **パスワード認証付き**（招待制の特別感演出）
- **チャット形式の相談予約ウィジェット**（右下に常時表示、名前→連絡先→カレンダー→時間帯→確認の対話式）

## LP2 - SNS広告用

- **ターゲット**: アラサー高給エリート会社員（年収800〜1200万）
- **訴求**: 感情・欲望（損失・機会・優位性）「知らないと損」「差がつく」
- **デザイン**: ダーク基調、スマホファースト、強コントラスト
- **構成**: Hero → Problem → Benefit → Track Record(圧縮) → Service → Voice → Limited Offer(カウントダウン) → FAQ → CTA
- **チャット形式の相談予約ウィジェット**（LP1と同等機能、ダークテーマ対応）

## 主な機能

- レスポンシブ対応（PC / タブレット / スマホ / 小型スマホ）
- LP1: パスワード認証ゲート（JS簡易版）
- LP1: VOICEセクション自動スクロールカルーセル
- LP2: 期間限定オファーのカウントダウンタイマー
- 共通: チャット形式の相談予約モーダル（右下ウィジェット）
- 共通: スクロールアニメーション（fade-in / fade-up）

## 技術仕様

- HTML / CSS / JavaScript（フレームワークなし）
- レスポンシブ対応（768px / 480px ブレークポイント）
- GitHub Pages でホスティング
- 画像: Unsplash フリー素材（仮）

## デプロイ

GitHub Pages で手動デプロイ（Settings > Pages > Deploy from a branch > `main`）
