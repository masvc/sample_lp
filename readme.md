## ページ構成

| パス | 内容 | 備考 |

| `/` | トップページ | LP1・LP2への導線 |

## LP1 - 営業カタログ

- **ターゲット**: アラフォー経営者（年収2000万〜）
- **訴求**: 理性・納得（実績・再現性・仕組み）
- **デザイン**: 緑・ゴールド・ホワイト基調、余白広め
- **構成**: Story → My Journey → Achievement → Voice → Support → CTA
- **パスワード認証付き**（招待制の特別感演出）

## LP2 - SNS広告用

- **ターゲット**: アラサー高給エリート会社員（年収800〜1200万）
- **訴求**: 感情・欲望（損失・機会・優位性）
- **デザイン**: ダーク基調、スマホファースト、強コントラスト
- **構成**: Hero → Problem → Benefit → Track Record → Service → Voice → Limited Offer → CTA
